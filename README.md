# Online store project

## PERN stack project of an online store, 
The admin can upload products and classify them, the user can easily find products and add them to cart.

### Technologies used:
- **Backend:** Node.js + Express.js;
- **Frontend:** React;
- **Database:** PostgreSQL;
- **CSS:** Bootstrap;

## Launch

**Server launch**
```
npm run dev
```

**Client launch**
```
npm start
```
